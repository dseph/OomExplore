OomExplore is a sample which is used to show the usage of Outlook Object Model (OOM).  This is not a tool. It is published as an open-source example of API usage. 
This example has a lot of unfinished code.  It can be used as example code and seeing how to navigate through OOM. Further code may or may not be added to it in the future.
